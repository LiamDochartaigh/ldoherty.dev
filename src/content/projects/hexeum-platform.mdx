## About Project

The Hexeum Interactive Platform evolved from the established Hexeum overlay marketplace to meet growing demand for dynamic streaming tools.

Currently processing over 1.5 million Twitch messages and events monthly, the platform serves 10k+ registered streamers with live chat overlays, social media widgets, community engagement and tipping features.

Unlike the traditional overlays, this platform provides interactive tools that integrate with existing software such as OBS or Streamlabs.

## Tech

- **Frontend:** Nuxt.js, Vue.js, Vuetify, TypeScript
- **Backend:** Node.js, Nitro, MongoDB
- **Real-time:** Socket.io, WebSockets, Twitch API
- **Tools:** Vite
