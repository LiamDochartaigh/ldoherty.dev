## About Project

These are custom mini-game widgets for Twitch streamers designed to boost engagement and revenue. The core innovation is integrating viewer actions like chatting, following, donating, and subscribing directly into interactive gameplay. By creating fun experiences where viewers can challenge each other or collaborate toward common goals, these widgets help streamers build stronger communities while increasing revenue potential. Built and deployed for Charming Streams.

## Tech

- [StreamElements Devkit](/projects/streamelements-devkit)
- JQuery
- TypeScript
- CSS / HTML / JavaScript
- Twitch API
